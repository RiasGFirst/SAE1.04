Project Reddit API

Quelque précision le projet Reddit

But de l'API

Nous essayons de rendre l'API de Reddit accessible à tous.
En particulier un des aspects qui est la recuperation de nimporte quel poste d'un subreddit (Pour archivage).

Github

Documentation

Nous demandons un acces token sur le site de l'API puis grace a une requete GET sur api.riascloud.fr/reddit/get/{subreddit}.
Nous obtenons les 25 derniers posts ou les nouveaux posts depuis notre derniere requete.
